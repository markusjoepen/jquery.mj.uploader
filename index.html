jquery.mj_uploader

Simple javascript enhanced, iframe based file uploader using PHP which displays uploaded images, videos and swf files after upload, and has the option to delete uploaded files (on doubleclick)

Basic Usage

$('#file').mj_uploader({dir:'uploads'});

Dependencies: jQuery, PHP >= 5.3

Demo

PropertyDefaultDescription
handler 'mj.uploader.php' path to the php file that handles the upload
dir '/' path to the upload directory.
directoryCheck true

checks if [dir] exists and if it has the writing permissions.

Helpful for development, set to "false" in production mode
width '100%'
height '100%'
fileTypes [] Array of allowed file extensions. By default (empty array), all extension are allowed. (e.g. ['jpg','gif','png']
fileName keep,auto "keep" keeps the original filename upon upload, "auto" generates a unique random filename
fileConflict keep, replace "keep" does not overwrite files of the same name in 'dir', but appends a [_int] suffix to the fileName (e.g. 'fileName_1.jpg') if it exists, "replace" replaces the existing file with the new one.
fileRemove false,true lets the user remove the file after the upload has completed. This will physically remove the file from your server's upload directory ('dir').
